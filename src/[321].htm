Ручная сортировка/группировка серий в плеере

Anton Polishchuk created an issue 2016-02-16

Сделать возможной вручную задавать параметры группировки серий в плеере. Задача касается категорий:

  • сериалы
  • мультсериалы
  • аниме
  • документальные фильмы

Проблема:

Сейчас видео автоматически группируются (схлопываются) по 10 штук. Это не всегда корректно отображает пользовательские ожидания. Некоторые серии сериалов заливаются в разных озвучках (от 1 до 5). Итого, вместо ожидаемых 10 серий пользователь может получить только 2 (по 5 вариантов озвучки на каждую).

Проблема усугубляется, если нужно посмотреть серию на втором десятке. Найти в два клика – только наугад.

    Чтобы определить серии с неправильным именованием (начинаются НЕ с числа), нужно выполнить следующий sql:

    SELECT t1.*, CONCAT('http://zerx.co/', p.id, '-', p.alt_name, '.html') AS url
FROM (
SELECT t.id_serial, GROUP_CONCAT(t.name) AS names, SUM(t.cnt) AS cnt FROM 
(SELECT id_serial, CAST(`name` AS UNSIGNED) AS ser_num, name, count(name) as cnt
FROM `series_video` 
WHERE name not like '0%'
GROUP BY id_serial, name
HAVING ser_num=0
ORDER BY id_serial
LIMIT 10000) as t
GROUP BY t.id_serial) as t1
LEFT JOIN dle_post p on t1.id_serial=p.id;

    Список URL на фильмы из результата SQL на zerx.co - в приложении
