Добавить генерацию метатегов в зависимости от выбранных пунктов фильтра

Dmitrii Prisacari created an issue 2016-04-11

Денис предоставил список условий и метатеги, которые должны генерироваться:

  • Если выбрано два и больше фильтра одного блока фильтров, на странице сгенерировать мета тег:

    <meta name="robots" content="NOINDEX,NOFOLLOW"/>

  • Если выбрано два и больше фильтров разных блоков фильтров, на странице сгенерировать мета тег

    <meta name="robots" content="NOINDEX,NOFOLLOW" />

  • Если выбран только один фильтр из блока любого блока на странице сгенерировать мета тег

    <meta name="robots" content="INDEX,FOLLOW"/>

  • На все страницы пагинации необходимо добавить внутри раздела HEAD мета тег

    <meta name="robots" content="noindex, follow" >

  • При использовании сортировки внутри HEAD помещается тег:

    <meta name="robots" content="noindex, nofollow" >
