Модуль загрузки видео вручную

yybitb created an issue 2016-12-12

Задача: Реализовать модуль загрузки видео вручную.

ref.

[12:11:16] Dmitrii Prisacari [kvertix php]: На выходных я переносил логику автоматизированной загрузки видео через граббер на новый сервер загрузки. Все свелось к одному скрипту. Если коротко:
1. скрипт проверяет, есть ли файлы, которые нужно скачать
2. скачивает файл к себе во временную директорию
3. копирует файл на одно из хранилищ
4. копирует файл на бекапный сервер
5. регистрирует файл в таблице cdn и отмечает в таблице сайта процентаж скачивания, равный 100%
6. удаляет файл из временной директории

Это половина дела загрузки видео. Вторая половина - это загрузка видео вручную. 
Раньше контентщики загружали через ftp, затем запускали модуль индексации, затем
копировали код файла в поля с сериями и т.д.
От этой логики мы сейчас будем отказываться в пользу загрузки файла через ajax (как 
это сделано в google docs, например). То есть, контентщик переносит видео-файл на 
страницу админки, файл загружается, ajax возвращает код cdn, который можно 
просто скопировать и вставить в поле с фильмом. 
Кроме поля для вставки файла к этой странице будет добавлен список с "историей 
загрузок".

Становится очевидно, что и при автоматической загрузке и при ручной, таблица с данными о загрузках на сайте должна иметь один формат. Записи о файлах, загруженных вручную, могут просто отличаться флагом, чтобы их было проще доставать для отображения в истории загрузок.

Но преред этим нужно будет сделать следующее. У нас есть на сайте 2 таблицы seasonvar_files и seazonvar. В этих двух таблицах фактически "размазана" информация по загрузке файлов с других сайтов. Нужно проанализировать таблицы и код, затем составить отчет о том, как объединить эти таблицы в одну и как эта таблица будет выглядеть. Может удастся какие-то поля даже сократить. Предлагаю результат назвать cdn_download.
С этой таблицей будут взаимодействовать как со стороны сайта, так и со стороны внешних серверов.

Со стороны сайта с таблицей будут взаимодействовать:
1. Граббер
2. Модуль загрузки вручную

Со стороны других серверов с таблицей будут работать скрипт автоматизированного скачивания файла (script/server/downloader/new-logic/download-file.php)
