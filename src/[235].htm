Связать модуль редактирования подборок с СЕО модулем

Dmitrii Prisacari created an issue 2016-08-01

На данный момент, если зайти в модуль редактирования подборок, во вспылвающем окне редактирования есть поле "Текст". Это поле должно хранить в себе описание подборки. Сейчас содержимое этого поля никак не выводится на страницу с подборкой. Соответсвенно, роль поля "Текст" не определено. Все описания подборок редактируются через СЕО модуль.

Это вызывает некоторые трудности. Например, в случае изменения URL подборки в модуле управления подборками, придется менять и URL в СЕО модуле. Это уже не совсем удобно.

Необходимо реализовать возможность редактирования всех данных о подборке в модуле управления подборками.

Предложение: так как работа с СЕО данными будет реализовываться в двух разных местах, для удобства можно было бы реализовать API СЕО модуля, через которое была бы возможность менять данные по URL.

Дополнение 1

  1. Нужно проверить, используются ли где-то поля "Постоянно", "От", "До". Если они не имеют особого значения, нужно убрать о них упоминания из интерфейса, кода и базы.

  2. На скрине открыто окно редактирования подборки топ250 кинопоиска. Почему-то отображается битая картинка.
